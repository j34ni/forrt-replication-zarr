# `paper/` — source paper PDF (not used in this replication)

The template expects a source paper PDF here, read by the `paper-analyst` agent to
seed Phase 1 (`00_paper_summary.md` + `01_quote.md`).

**This replication is question-rooted, not paper-rooted** — there is no upstream
paper making the claim, so no PDF lives in this directory. The claim under test was
surfaced from a public README rather than a publication; see
[`nanopubs/drafts/00_paper_summary.md`](../nanopubs/drafts/00_paper_summary.md) for
the primary citable source (the `zarr-datafusion-search` README by Development Seed)
and the verbatim claim sentence it contains. The chain therefore opens with a **PICO
question** (`01_pico.md`) rather than a Quote-with-comment — see
[`docs/chain-decision-tree.md`](../docs/chain-decision-tree.md) for why a question-rooted
chain takes this shape.

If a future replication added to this repository *is* paper-rooted, drop its PDF here
following the naming convention `{first-author}-{year}.pdf` (e.g. `soroye-2020.pdf`)
and reference it from `index.md`.
